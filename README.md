# Bio-Scout DK 🌿🤖

**Bio-Scout DK** is a high-tech Citizen Science application designed to gamify nature exploration in Denmark.

## Vision
To bridge the gap between complex biodiversity data collection and modern outdoor lifestyles using a sleek, tech-first approach.

## Current Features (Alpha)
- [x] High-tech Dark Mode UI
- [x] Live Camera Access (Scanner Interface)
- [x] Dynamic Mission Board
- [x] Responsive Design for Mobile

## Tech Stack
- HTML5 / Tailwind CSS
- JavaScript (Vanilla)
- Deployed via GitHub Pages
